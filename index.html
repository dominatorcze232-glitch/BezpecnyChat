Zachycená chyba Firebase:

Žádné chyby nezachyceny.

1. Nastavení anonymního přihlášení

Otevři Firebase Console -> Authentication -> Sign-in method -> Klikni na "Add new provider" -> Vyber "Anonymous" a zapni "Enable".

2. Nastavení Cloud Firestore databáze

Otevři Firebase Console -> Firestore Database -> Klikni na "Create database". Nastav lokaci a v záložce "Pravidla" (Rules) se ujisti, že jsou povolené zápisy. (Nebo přepni do Test Mode).
